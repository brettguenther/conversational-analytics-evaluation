# Changelog

## [0.2.0] - 2025-11-17

### Added

-   Support for multiple Looker explores in the `looker` command. The CLI now accepts multiple `--model-explore` arguments in the format `model_name/explore_name`.

### Changed

-   The `--looker-model` and `--looker-explore` CLI options have been removed and replaced by `--model-explore`.
